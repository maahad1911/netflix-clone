### Common Issues & Solutions

**Page Not Loading**
- Use local server (Python or Node)
- Clear browser cache
- Check file paths

**Images Not Showing**
- Verify image file paths
- Check file names and extensions
- Use absolute URLs for external images

**Videos Not Playing**
- Ensure MP4 format
- Check video file path
- Verify browser support
